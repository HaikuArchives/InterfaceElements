Preferences

Resource Editor Preferences

    Use Workspace

      You can use a dedicated workspace for Interface Elements. When this setting is active, Interface Elements will switch to the given workspace and will use only that workspace. Range is [0 - maximum workspaces-1]. This setting will be active next time you start the application.

    Make Backup

      When you save an existing resource file (e.g. "Resource File"), a backup file will be created ("Resource File backup").

    Open Resource When Created

      When you create a resource by dropping it on the resource window, the resource will be open for editing if this setting is turned on.

    Allow Opening More Editor Windows For The Same Resource

      Select whether you want to open another editor window if one is already open, or just activate the existing editor window.

    External hex-editor

      Drag your favourite hex editor (e.g. DiskProbe) here. This will be used by the 'Data Editor' when you want to edit a resource as hex (press "Edit as Hex").

    External text-editor

      Drag your favourite text editor (e.g. StyledEdit) here. This will be used by the 'Data Editor' when you want to edit a resource as text (press "Edit as Text").

View Editor Preferences

    Confirm View Parent Change

      If set, you will be asked to confirm if a view can attach to another parent.

    Default Snap

      The default snap value when the view editor opens.
